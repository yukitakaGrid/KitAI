## Version

OpenAI : gpt-4-turbo-preview


## How to use
事前に必要なライブラリをインストールしておきます。
```
pip install openai
pip install discord.py
```


## Reference
このbotはedit modeとcommand modeの2種類のモードが存在します。デフォルトはcommand modeです。
